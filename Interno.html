EJEMPLO DE INTERNAL

CSS es el acrónimo de Cascading style sheets, que en castellano se traduce por hojas de estilo en cascada. Desarrollado por el W3C en 1996, CSS es un mecanismo complementario al lenguaje HTML que permite indicarle al navegador el estilo con el qué debe de mostrar los distintos elementos que forman parte de la página web a visualizar. Mientras que el HTML se centra en indican al navegador que ciertas partes la pagina son títulos, subtítulos, párrafos, enlaces, etc. El CSS complementa esa información para que el tipo de fuente, tamaño, color o alineación con el que se muestre cada elemento sea el correcto.
